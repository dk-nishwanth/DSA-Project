### Bellman-Ford Visualizer
- **Interactive edge relaxation** with step-by-step animation
- **Distance tracking** for all vertices from source
- **Visual graph representation** with directed edges
- **Iteration counter** showing V-1 relaxation rounds
- **Negative cycle detection** capability
- **Color-coded states**: active edges, relaxed nodes, updated distances

### Maze Solver Visualizer
- **5√ó5 grid maze** with walls and paths
- **Real-time path exploration** showing backtracking
- **Color-coded cells**: white (path), black (wall), yellow (exploring), green (solution)
- **Start (S) and End (E) markers** clearly visible
- **Path length tracking** during exploration
- **Automatic backtracking** when hitting dead ends
- **Success indicator** when path is found

### Generate Parentheses Visualizer
- **Tree exploration visualization** showing decision tree
- **Real-time constraint validation** (open < n, close < open)
- **Valid combinations display** as they're found
- **Catalan number calculation** showing expected results
- **Nested path display** with indentation
- **Open/close bracket counters** for each node
- **Completion markers** for valid strings
